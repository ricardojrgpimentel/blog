Caso estejam habituados a usar o vosso rato sem aceleração, ao utilizarem o macOS, vão deparar-se com este problema.

Por defeito o sistema acelera a velocidade de deslocação do rato em movimentos rápidos, e diminui a mesma em movimentos lentos.

Esta configuração torna o movimento do rato imprevisivel e um pouco irritante, mas com este comando tudo se resolve:

```bash
defaults write .GlobalPreferences com.apple.mouse.scaling -1
```

O número negativo no fim do comando faz com que a aceleração seja desativada, caso pretendam ligar novamente esta abominação, basta executar o mesmo comando mas com um valor positivo, à vossa preferência de aceleração.

Caso o resultado não seja imediato, reiniciar o computador é sempre a solução.
