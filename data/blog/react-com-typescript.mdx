## Types

Antes de iniciarmos com exemplos, devemos de ter em conta que quando declaramos um _type_, não estamos a declarar um novo _type_, mas sim a dar um novo nome a um _type_.

A lista de _types_ encontra-se [aqui](https://www.typescriptlang.org/docs/handbook/2/everyday-types.html), ao qual é dado o nome de **Everyday Types**.

É também importante saber quando utilizar os _types_ e as _interfaces_, pois são bastante parecidas, de uma forma resumida uma _interface_ representa
o formato dos dados e os _types_ os tipos de dados.

A declaração de variáveis é bastante simples, basta apenas declarar a mesma, seguido de `:` e o tipo de variável:

```javascript
type Pessoa = {
  //tipos primitivos
  nome: string
  idade: number
  casado: boolean
  //valores literais
  naturalidade: 'Portugal' | 'Espanha' | 'Outra'
  //objetos
  saude: {
    //Arrays com vários tipos
    vacinas : string[] // ou também Array<string>
    //funções
    podeVacinar : (idade: number) => boolean
  },
  //opcional
  telefone?: number
  //eventos de HTML
  submeter: (evento: React.FormEvent<HTMLInputElement>) => void
}
```
